# Threat Detection Rules

This repository contains example detection rules and investigation guides used by security operations teams to identify suspicious activity within enterprise environments.

The goal of this project is to demonstrate detection engineering concepts including:

- Security event monitoring
- Behavioral detection logic
- Threat hunting methodology
- MITRE ATT&CK mapping

All examples are generic and intended for educational demonstration purposes only.

## Topics Covered

• Impossible travel authentication detection  
• Suspicious PowerShell activity  
• Privilege escalation detection  
• Mass file download activity  

## Detection Workflow

Each detection example includes:

1. Threat description  
2. Detection logic  
3. Example indicators  
4. Response actions  
5. MITRE ATT&CK mapping
